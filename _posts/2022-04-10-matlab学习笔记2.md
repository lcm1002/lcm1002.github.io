## 矩阵运算的应用——线性方程组的求解

```matlab
%例1.2.25
A=[5 6 0 0 0
   1 5 6 0 0
   0 1 5 6 0
   0 0 1 5 6
   0 0 0 1 5];
B=[1,0,0,0,1]';
R_A=rank(A);
X=A\B;
R_A,X
```

> R_A = 5
>
> X = 5×1
>     2.2662
>    -1.7218
>     1.0571
>    -0.5940
>     0.3188

```matlab
%另一种解法
A=[5 6 0 0 0
   1 5 6 0 0
   0 1 5 6 0
   0 0 1 5 6
   0 0 0 1 5];
B=[1,0,0,0,1]';
C=[A,B];   %构造增广矩阵
R=rref(C); %将C化成最简行
R
```

> R = 5×6
>     1.0000         0         0         0         0    2.2662
>          0    1.0000         0         0         0   -1.7218
>          0         0    1.0000         0         0    1.0571
>          0         0         0    1.0000         0   -0.5940
>          0         0         0         0    1.0000    0.3188
